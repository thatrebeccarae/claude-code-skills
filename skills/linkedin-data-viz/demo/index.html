How It Works

Four steps to your LinkedIn story

1
Export
Request your data archive from LinkedIn Settings. Takes ~24 hours.
2
Install
Copy the skill to your Claude Code skills directory.
3
Run
Tell Claude "analyze my LinkedIn data" and follow the wizard.
4
Share
Optionally sanitize your data and publish a demo version.
Installation

Get started in 30 seconds

# Clone the repo
git clone https://github.com/thatrebeccarae/claude-code-skills.git

# Copy skill to your Claude Code skills directory
cp -r claude-code-skills/skills/linkedin-data-viz ~/.claude/skills/

# Then in Claude Code, just say:
"Analyze my LinkedIn data export"
Features

Built for privacy and polish

9 Interactive Visualizations
D3.js force graphs, Chart.js charts, and CSS layouts. Drag nodes, filter messages, hover for details.
Dark Theme
Carrier Dark design system — glass panels, teal/lime accents, JetBrains Mono for data.
Privacy-Safe Publishing
Built-in sanitizer replaces real names and companies with plausible fakes while preserving all patterns.
Guided Wizard
Step-by-step onboarding — from CSV location to generated HTML. No coding required.
